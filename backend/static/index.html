How it works

Install, flag, done.

Install the extension

One click from the Chrome Web Store. No sign-up. A random anonymous ID is generated on your machine and never leaves it.

Flag AI slop when you see it

A small "AI slop" button appears next to YouTube's like/dislike buttons. Hit it when something looks generated. Hit it again to undo.

The label kicks in

Once 3 independent users flag the same video, the button turns orange for everyone. You'll know before you waste your time.

Details

What you'd want to know

No account, no email

Your install gets a random UUID. That's the only identifier. We don't know who you are and don't want to.

Fully open source

Extension, Rust backend, Postgres schema, threshold logic — all on GitHub. Fork it, audit it, run your own instance.

Works on every video

The button appears next to like/dislike on any YouTube watch page. If you can watch it, you can flag it.

Undo at any time

Reported by accident? Click the button again. Your report is removed from the count instantly.

FAQ

Common questions

How does YouTube AI Detector work?

Install the Chrome extension with one click. When you see an AI-generated video, click the "AI slop" button next to YouTube's like/dislike buttons. When 3 independent users flag the same video, the button turns orange for everyone.

Is it free?

Yes, completely free and open source under the MIT license. No account, no email, no tracking. The extension, backend, and all code are on GitHub.

Does it track my data or require an account?

No. Your install gets a random anonymous UUID that stays on your device. There is no account, no email, no tracking. The developers cannot identify you.

Can I undo a report?

Yes. Click the report button again and your report is removed from the count instantly.

My video was reported by mistake. What can I do?

Reach out to lukasdzn1@gmail.com and we'll look into it. We take false reports seriously.
